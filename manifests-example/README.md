# Example Winget Manifests

⚠️ **These are example manifests for reference only, not the authoritative package definition.**

The official winget package for Toasty must be submitted to and maintained in the [microsoft/winget-pkgs](https://github.com/microsoft/winget-pkgs) repository.

These example manifests are provided to:
- Demonstrate the proper structure for winget package submission
- Serve as a template for creating new version manifests
- Show the verified SHA256 hashes for v0.6 release assets

**Do not treat these files as the authoritative package state.** They may become outdated as new versions are released.

## Submitting to winget-pkgs

See [WINGET.md](../WINGET.md) for complete submission instructions.
